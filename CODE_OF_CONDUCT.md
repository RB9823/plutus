# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Standards

- Be respectful and constructive.
- Focus critiques on ideas and implementation details.
- Assume good intent, ask clarifying questions, and avoid personal attacks.

## Enforcement

Maintainers may remove comments, issues, commits, code, wiki edits, and other contributions that are not aligned with this code of conduct.

## Reporting

Report conduct issues to: `maintainers@plutus.dev`.
